# CT Demand Marketing Dashboard

**Live:** https://chile-ct.github.io/ct-demand-marketing/  
**Repo:** https://github.com/chile-ct/ct-demand-marketing  
**Localhost:** http://localhost:8000 (auto-starts on Mac login via LaunchAgent)

## What's in this folder

| File/Folder | Purpose |
|---|---|
| `src/index.html` | **Edit here** — JSX source for Marketplace dashboard |
| `dist/index.html` | Compiled output (auto-generated by `node build.js`) |
| `index.html` | GitHub Pages copy (= dist/index.html) |
| `app-growth.html` | App Growth dashboard (separate, auto-updated Mon 10:30) |
| `data.json` | Data file for app-growth.html |
| `build.js` | Compiles JSX → plain JS (no browser Babel) |
| `scripts/update_marketplace.py` | **Main BQ updater** — patches src + builds + pushes |
| `scripts/run_update_marketplace.sh` | Cron wrapper for above |

## Auto-update schedule

| When | What | How |
|---|---|---|
| Daily 11:00 SA | Marketplace data (DAU/MAU/DwL/Lead) | Local cron → `run_update_marketplace.sh` |
| Daily 11:00 SA (UTC+7) | Same, via cloud | GitHub Actions (`update-dashboard.yml`) |
| Mon 10:30 SA | App Growth data | Local cron → `claude -p "..."` |

## Make a change

```bash
# 1. Edit source
nano src/index.html

# 2. Build
node build.js

# 3. Preview locally
open http://localhost:8000

# 4. Push
cp dist/index.html index.html
git add src/index.html dist/index.html index.html
git commit -m "feat: ..."
git push
```
